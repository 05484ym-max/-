חיבור ל-GitHub

זה יאפשר לכם ללחוץ "פרסום ישירות לאתר" ולעדכן את החנות בלי להוריד/להעלות קבצים בעצמכם.

1. ב-GitHub: לחצו על תמונת הפרופיל ⟵ Settings ⟵ Developer settings ⟵ Personal access tokens ⟵ Fine-grained tokens ⟵ Generate new token
2. תחת "Repository access" בחרו Only select repositories ואת הריפו שלכם בלבד
3. תחת "Permissions" ⟵ Contents ⟵ בחרו Read and write
4. Generate token, והעתיקו אותו לכאן (הוא נשמר רק בדפדפן שלכם, לא נשלח לשום מקום חוץ מ-GitHub)

מבנה הריבועים באתר

כאן בונים את כל מבנה הריבועים: מוסיפים ריבוע חדש לדף הבית, או תת-ריבוע בתוך כל קטגוריה קיימת (בכל עומק) — למשל "כשרים" ו"לא כשרים" בתוך "סלולר".

מוצרים בקטגוריה

תפריט צדדי (המבורגר)

השורות שמופיעות כשלקוח פותח את תפריט ההמבורגר באתר. לכל שורה יש שני סוגים אפשריים: קישור (יוצא מהאתר - וואטסאפ, ווייז, טלפון...) או דף מידע (נפתח בתוך האתר עצמו, עם טקסט חופשי - כמו "שעות פתיחה" או "אודות").

בחרו מוצר לעריכה מהרשימה, או לחצו "מוצר חדש" כדי להתחיל ⬅️
